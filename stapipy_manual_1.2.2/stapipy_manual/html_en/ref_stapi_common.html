Common Functions and Classes¶

Common Functions¶

initialize¶

stapipy.initialize()¶

Initialize StApi.

This function must be called before using other StApi functions. Call terminate() to release the resource of StApi when it is no longer needed.

terminate¶

stapipy.terminate()¶

Uninitialize StApi.

Call this function to release resources and de-initialize the StApi.

get_version¶

stapipy.get_version(bool as_text=True)¶

Acquire version of the running StApi. If the version is returned as numeric: bit 31-24: major version, bit 23-16: minor version, bit 15-0: subminor version.

Parameters

as_text (bool) – if True, return the version as string. Otherwise, return the version as numeric int.

Returns

Version of the StApi.

Return type

str (if as_text is True) or int (if as_text is False).

Common Exception¶

PyStError¶

class stapipy.PyStError(message)¶

Bases: Exception

This class propagates exceptions raised by the StApi and GenICam.

Example:

try:
    # code
except PyStError as st_error:
    print(st_error)
except Exception as other_exception:
    print(other_exception)

It is also possible to only catch Exception and check the type of the exception.

Example:

try:
    # code
except Exception as exception:
    print(type(exception), exception)

PyStRegisteredCallback¶

class stapipy.PyStRegisteredCallback(callback_func=None, user_data=None)¶

Bases: object

This class stores information of registered callback.

callback_type¶

PyStRegisteredCallback.callback_type: EStCallbackType

Property: callback type (only for callback triggered by StApi). If the callback is triggered by GenApi node, None will be returned.

Getter

Returns callback type.

Return type

EStCallbackType

data¶

Property: Data depending on the EStCallbackType.

Getter

Returns data related to the callback.

Return type

dict

error¶

Property: a tuple containing error code (int) and description (str) if the callback is triggered due to an error. If there is no error, a tuple of (None, None) is returned. In case that error code is not available, a tuple of (None, str) is returned.

Getter

Returns error code (int) and description (str), or (None, None)

Return type

tuple of (int, str) or (None, None)

is_released¶

PyStRegisteredCallback.is_released: bool

Property: Flag indicating that the resource used for the callback has been released.

Getter

Returns True if the callback resource has been released, False otherwise.

Return type

bool

module¶

Property: StApi class that triggered the callback. The class is one of the following: PyStSystem, PyStInterface, PyStDevice, PyStDataStream, PyStStreamBuffer, PyStVideoFiler. If the callback is triggered by StApi node, None will be returned.

Getter

Returns registered module that triggered the callback, or None.

Return type

PyStSystem or PyStInterface or PyStDevice or PyStDataStream or PyStStreamBuffer or PyStVideoFiler or None

node¶

Property: Node that triggers the callback. If the callback is not triggered by a GenApi node, None will be returned.

Getter

Returns registered node that triggered the callback, or None.

Return type

PyNode or None

pause(self, value) → bool¶

Temporarily stop triggering the callback.

Parameters

value (bool) – True to pause. False to un-pause.

Returns

True if pause command is successfully processed. False otherwise.

Return type

bool

release(self)¶

Release the resource used for the callback. Once released, the callback cannot be triggered anymore.
