EStBayerInterpolationMethod¶

class stapipy.EStBayerInterpolationMethod(value)¶

Bases: enum.Enum

Enumeration: Bayer interpolation method

Off = 0

Off. Converts a Bayer image to a color image with the color of each pixel’s filter without color interpolation.

NearestNeighbor = 1

Nearest neighbor 1. Copies neighboring pixel values. The position of the copied pixel is different from that of the Nearest neighbor 2.

NearestNeighbor2 = 2

Nearest neighbor 2. Copies neighboring pixel values. The position of the copied pixel is different from that of the Nearest neighbor 1.

BiLinear = 3

Bilinear 1. Uses the average of neighboring pixel values. The processing load is lower than BiLinear 2 and 3, but the false color is conspicuous.

BiLinear2 = 4

Bilinear 2. Uses the average of neighboring pixel values. Fake edge coloring is less than BiLinear 1, but the processing load is higher.

BiCubic = 5

BiCubic. Uses the surrounding 4x4 pixels.

Mono = 6

Mono. Convert bayer image to monochrome image without performing color interpolation processing.

BiLinear3 = 7

Bilinear 3. Uses the average of neighboring pixel values. Fake edge coloring is less than BiLinear 2, but the processing load is higher.

Count = 8

Count of enumerator.
