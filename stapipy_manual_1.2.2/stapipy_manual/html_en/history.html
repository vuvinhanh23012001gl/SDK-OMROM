Release Notes¶

Release: 2024.05.31 (Version: 1.2.2)¶

  • Change: SentechSDK v1.2.2 must be installed.

  • Added support for GenApi3.4.2.

  • Add sample programs:

    • user_memory.py

  • Added support for Python 3.10.

Release: 2023.04.14 (Version: 1.2.1.1)¶

  • Add new classes as follows:

    • PyStDeviceList class

    • PyStDataStreamList class

    • PyStSystemList class

  • Add datastream property to PyStStreamBuffer class.

  • Add sample programs:

    • multiple_cameras.py

    • multiple_systems.py

Release: 2022.12.16 (Version: 1.2.1)¶

  • Change: SentechSDK v1.2.1 must be installed.

  • copy_settings method added to PyStConverter, PyStFilter, PyStFiler

  • A callback was added when the list was updated in UpdateInterfaceList and UpdateDeviceList.

  • Add StApi IP Filter: Level adjustment function

  • Add get_value_f(index) method to PyStPixelComponentValue class.

  • Add interface_type property to PyStInterface class.

  • Add VirtualFrameGrabber=16 to EStInterfaceType. (Windows only)

  • The following item is added to the EStPixelComponent: Mono,R,G,B,Gr,Gb,Y

  • The following item is added to the EStPixelFormatNamingConvention: RGB10,RGB12,RGB14,RGBa10,RGBa12,RGBa14,RGBa16.

  • Add new enumeration types as follows:

    • EStMemoryInitialization

    • EStDefectivePixelDetectionAveragingArea

    • EStDefectivePixelDetectionThresholdTarget

  • Add methods to PyStImage class, as follows:

    • create_buffer(width, height, pixel_format, initialize_memory) method

    • clear_component_data(pixelcomponent) method

    • import_component_data(pixelcomponent, image) method

    • export_component_data(pixelcomponent) method

    • is_same_image(image) method

    • store_computed_image(operation_type, pixel_format, image1, image2) method

  • Add properties to PyStPixelFormatInfo class, as follows:

    • is_polarization property

    • order_of_polarized_filters property

  • Add compression_level property to PyStStillImageFiler class.

  • Add methods and properties to PyStDefectivePixelDetectionFilter class, as follows:

    • get_threshold(threshold_target) method

    • set_threshold(threshold_target, value) method

    • defective_pixel_detection_averaging_area property

    • peripheral_area_size property

    • maximum_pixel_count property

  • For GenApi3.3 support.

    • Add get_length(verify=False) method to PyIRegister class.

    • Add get_address(verify=False) method to PyIRegister class.

    • Add get_max_length(verify=False) method to PyIString class.

  • For GenApi3.4 support.

    • Add new enumeration type as follows: EGCCallbackSuppressMode.

    • Add set_suppress_callback_mode(suppress_mode) method to PyNodeMap class.

  • Added support for Python 3.9.

Release: 2022.09.16 (Version: 1.1.2.4)¶

  • Added PyIPort class to support GenApi::IPort.

Release: 2022.05.20 (Version: 1.1.2.3)¶

  • For SentechSDK v1.1.2 Update6 support.

    • The following items have been added to the EStSystemVendor: Matrox, TeledyneDALSA, AvalData.

Release: 2021.12.03 (Version: 1.1.2.2)¶

  • For SentechSDK v1.1.2 Update5 support.

    • The following item is added to the EStBayerInterpolationMethod: BiLinear3.

  • Added support for Python 3.8.

Release: 2021.10.08 (Version: 1.1.2.1)¶

  • For SentechSDK v1.1.2 Update4 support.

    • The following pixel format is added to the EStPixelFormatNamingConvention: CYCbCr420_8_YY_Cb_Cr_Planar.

    • The following item is added to the EStVideoFileFormat: WebM. (Windows only)

    • The following item is added to the EStVideoFileCompression: VP8. (Windows only)

  • Add stapipy.get_filter_info(filter_type) function.

  • Change: minor fixes of docstring (document).

Release: 2020.07.31 (Version: 1.1.2)¶

  • Change: SentechSDK v1.1.2 must be installed.

  • Add StApi IP Filter: Defective Pixel Detection.

  • Add StApi IP Filter: Image Averaging

  • Add StApi IP Filter: Graph Data.

  • Add static method create_from_data to PyIStImage to allow generation of PyIStImage from bytearray data.

  • Add elements to the following enumeration types to support polarization sensors.

    • EStPixelFormatNamingConvention

    • EStPixelComponent

    • EStPixelColorFilter

Release: 2019.08.30 (Version: 1.0.8)¶

  • No changes from last version. Confirmed support for SentechSDK v1.0.8.

Release: 2019.05.31 (Version: 1.0.7)¶

  • Remove unimplemented Graph enumerator from EStFilterType.

  • Change: minor fixes of docstring (document).

Release: 2019.05.17 (Version: 1.0.7rc3)¶

  • Add StApi IP functionality as follows:

    • Converter: Pixel format, reverse.

    • Filter: Balance ratio, chroma suppression, color transformation, edge enhancement, flat field correction, Gamma correction, noise reduction, SN measurement.

    • Filer: Still image and video.

  • Add samples:

    • grab_ip_opencv.py

    • save_and_load_image.py

    • save_video.py

    • singleconverter_opencv.py

    • singlefilter_opencv.py

Release: 2019.04.15 (Version: 1.0.7rc2)¶

  • Add supported environments:

    • Windows: Python 3.6.x 32 bit and 64 bit.

    • Linux: Python 3.6.x 32 bit and 64 bit.

    • MacOS 10.13: Python 3.6.x 64 bit.

  • Fix an issue: in certain cases resource of PyStImage is not properly released when PyStImage instance is destroyed.

  • Change: update comments of sample program.

  • Change: update reference structure of this document.

Release: 2019.04.11 (Version: 1.0.7rc1)¶

  • Supports transport layer functions (StApi_TL).

  • Supported environments:

    • Windows: Python 3.7.x 32 bit and 64 bit.

    • Linux: Python 3.7.x 32 bit and 64 bit.

    • MacOS 10.13: Python 3.7.x 64 bit.
