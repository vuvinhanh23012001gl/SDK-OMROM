リリースノート¶

リリース: 2024.05.31 (バージョン: 1.2.2)¶

  • SentechSDK v1.2.2が必要となります。

  • GenApi3.4.2に対応しました。

  • サンプルプログラム追加：

    • user_memory.py

  • Python3.10に対応しました。

リリース: 2023.04.14 (バージョン: 1.2.1.1)¶

  • PyStDeviceListクラス, PyStDataStreamListクラス, PyStSystemListクラスを追加しました。

  • PyStStreamBufferクラスに datastreamプロパティを追加しました。

  • サンプルプログラム追加：

    • multiple_cameras.py

    • multiple_systems.py

リリース: 2022.12.16 (バージョン: 1.2.1)¶

  • SentechSDK v1.2.1 が必要となります。

  • PyStConverterクラス, PyStFilterクラス, PyStFilerクラスにcopy_settingsメソッドを追加しました。

  • UpdateInterfaceList, UpdateDeviceListでリストが更新された際にコールバックを追加しました。

  • StApi IPフィルターに レベル調整機能 を追加しました。

  • PyStPixelComponentValueクラスにget_value_f(index)メソッドを追加しました。

  • PyStInterfaceクラスにinterface_typeプロパティを追加しました。

  • EStInterfaceType列挙型に VirtualFrameGrabber=16 を追加しました。(Windows版のみ)

  • EStPixelComponent列挙型にMono,R,G,B,Gr,Gb,Yを追加しました。

  • EStPixelFormatNamingConvention列挙型にRGB10,RGB12,RGB14,RGBa10,RGBa12,RGBa14,RGBa16を追加しました。

  • 以下の列挙型を追加しました。

    • EStMemoryInitialization

    • EStDefectivePixelDetectionAveragingArea

    • EStDefectivePixelDetectionThresholdTarget

  • PyStImageクラスに以下を追加しました。

    • create_buffer(width, height, pixel_format, initialize_memory)メソッド

    • clear_component_data(pixelcomponent)メソッド

    • import_component_data(pixelcomponent, image)メソッド

    • export_component_data(pixelcomponent)メソッド

    • is_same_image(image)メソッド

    • store_computed_image(operation_type, pixel_format, image1, image2)メソッド

  • PyStPixelFormatInfoクラスに以下を追加しました。

    • is_polarizationプロパティ

    • order_of_polarized_filtersプロパティ

  • PyStStillImageFilerクラスにcompression_levelプロパティを追加しました。

  • PyStDefectivePixelDetectionFilterクラスに以下を追加しました。

    • get_threshold(threshold_target)メソッド

    • set_threshold(threshold_target, value)メソッド

    • defective_pixel_detection_averaging_areaプロパティ

    • peripheral_area_sizeプロパティ

    • maximum_pixel_countプロパティ

  • GenApi3.3対応以下のメソッドを追加しました。

    • PyIRegisterクラス : get_length(verify=False)メソッド

    • PyIRegisterクラス : get_address(verify=False)メソッド

    • PyIStringクラス : get_max_length(verify=False)メソッド

  • GenApi3.4対応以下を追加しました。

    • EGCCallbackSuppressMode列挙型

    • PyNodeMapクラス : set_suppress_callback_mode(suppress_mode)メソッド

  • Python3.9に対応しました。

リリース: 2022.09.16 (バージョン: 1.1.2.4)¶

  • GenApi::IPortに対応するため、PyIPortクラスを追加しました。

リリース: 2022.05.20 (バージョン: 1.1.2.3)¶

  • SentechSDK v1.1.2 Update6に対応するため、以下を追加しました。

    • EStSystemVendor列挙型にMatrox, TeledyneDALSA, AvalDataを追加

リリース: 2021.12.03 (バージョン: 1.1.2.2)¶

  • SentechSDK v1.1.2 Update5に対応するため、以下を追加しました。

    • EStBayerInterpolationMethod列挙型にBiLinear3を追加

  • Python3.8に対応しました。

リリース: 2021.10.08 (バージョン: 1.1.2.1)¶

  • SentechSDK v1.1.2 Update4に対応するため、以下を追加しました。

    • EStPixelFormatNamingConvention列挙型にCYCbCr420_8_YY_Cb_Cr_Planarを追加

    • EStVideoFileFormat列挙型にWebMを追加 (Windows版のみ)

    • EStVideoFileCompression列挙型にVP8を追加 (Windows版のみ)

  • stapipy.get_filter_info(filter_type)関数を追加しました。

  • ドキュメント(docstring)を修正しました。

リリース: 2020.07.31 (バージョン: 1.1.2)¶

  • SentechSDK v1.1.2が必要となります。

  • StApi IPフィルターに画素欠検出機能、画像平均機能、グラフ機能を追加しました。

  • bytearray データから、PyIStImage を作成できるように、 PyIStImageにcreate_from_data関数を追加しました。

  • 偏光センサーに対応するため、以下の列挙型に要素を追加しました。

    • EStPixelFormatNamingConvention

    • EStPixelComponent

    • EStPixelColorFilter

リリース: 2019.08.30 (バージョン: 1.0.8)¶

  • 変更なし。SentechSDK v1.0.8動作確認済みのバージョン。

リリース: 2019.05.31 (バージョン: 1.0.7)¶

  • 未実装のGraph列挙子を EStFilterType から削除しました。

  • ドキュメント(docstring)の修正。

リリース：2019.05.17 (バージョン1.0.7rc3)¶

  • 下記のStApi IPの画像処理機能を追加します：

    • コンバーター：ピクセルフォーマット、反転。

    • フィルター：バランスレート、クロマサプレス、色変換、エッジ強調、フラットフィールド補正、ガンマ補正、ノイズ軽減、SN測定。

    • ファイラー：静止画、動画像。

  • サンプル追加：

    • grab_ip_opencv.py

    • save_and_load_image.py

    • save_video.py

    • singleconverter_opencv.py

    • singlefilter_opencv.py

リリース：2019.04.15 (バージョン1.0.7rc2)¶

  • 対応する環境を追加しました：

    • Windows: Python 3.6.x 32ビットと64ビット。

    • Linux: Python 3.6.x 32ビットと64ビット。

    • MacOS 10.13: Python 3.6.x 64ビット。

  • PyStImageのインスタンスを解放する際、リソースを解放されていない不具合を修正しました。

  • いくつかサンプルプログラムのコメントを更新しました。

  • 使いやすくために手順書のリファレンス構造を変更しました。

リリース：2019.04.11 (バージョン1.0.7rc1)¶

  • Transport Layer機能（StApi_TL）に対応します。

  • 対応する環境：

    • Windows: Python 3.7.x 32ビットと64ビット。

    • Linux: Python 3.7.x 32ビットと64ビット。

    • MacOS 10.13: Python 3.7.x 64ビット。
